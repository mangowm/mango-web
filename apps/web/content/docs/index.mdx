## What is MangoWC?

MangoWC is a dynamic Wayland compositor based on [dwl](https://codeberg.org/dwl/dwl/).

## Feature Highlights

- Excellent xwayland support
- Tag-based workspaces
- Smooth animations
- Excellent input method support
- Flexible window layouts
- Rich window states
- External configuration
- Scratchpad and overview
- IPC support
- Window effects
